MiAntivirus

MiAntivirus é uma interface gráfica para o ClamAV, que permite verificar seu computador em busca de vírus e atualizar a base de dados de definições de forma prática e intuitiva, tudo por meio de uma interface amigável.

Recursos

  • Detectar PUA (Aplicativo Potencialmente Indesejado)
  • Varredura heurística
  • Verifica arquivos ocultos
  • Verifica arquivos e pastas simbólicas
  • Verifica arquivos compactados
  • Verifica arquivos de e-mail

No MiAntivirus você pode adicionar uma lista de arquivos e pastas que deseja ignorar durante o escaneamento, como também definir um tamanho máximo em MB para escaneamento dos arquivos.
