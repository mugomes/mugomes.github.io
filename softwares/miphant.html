MiPhant

O MiPhant é desenvolvido para rodar os scripts PHP através de um navegador de Internet com janelas personalizadas, ele executa o servidor PHP embutido com portas aleatórias. Dessa forma, você pode executar vários aplicativos ao mesmo tempo sem que eles interfiram uns nos outros.

Além disso, o MiPhant também faz download automático do PHP compilado de forma estática (não depende de bibliotecas externas), graças ao excelente projeto static-php-cli.

Se precisar de uma extensão específica, você pode compilar o PHP para Linux e Windows com as extensões necessárias.

Idiomas
No MiPhant você pode desenvolver aplicativos em vários idiomas, você só precisa criar um arquivo de idioma no formato json na pasta langs e adicionar as traduções. Ao executar o aplicativo em um computador com um idioma diferente, o MiPhant detectará o idioma e obterá a tradução por meio do arquivo de tradução.

Perguntas Frequentes

Para que serve o MiPhant?

O MiPhant é um software portátil desenvolvido para navegar em sistemas PHP, dessa forma você pode desenvolver desde projetos pessoais até empresariais.

Posso desenvolver software proprietário e livres?

Você pode desenvolver softwares proprietários e livres

Posso ofuscar o código do meu software?

Sim. Softwares proprietários podem ter seu código ofuscado, para isso você pode trabalhar com ofuscadores, recomendamos o yakpro para ofuscação do PHP e JavaScript Obfuscator para ofuscação de códigos JavaScript. Você também pode utilizar softwares pagos para isso ou usar técnicas mais simples como usar o base64, mas lembre-se, quanto mais simples a técnica de ofuscação for, mais fácil será desofuscá-lo.

Posso vender licenças para meu software?

Se o seu software for proprietário você pode vender sua licença como desejar, sob licenças, limitação de uso de computadores e usuários, licença vitalícia, entre outras formas.

Posso desativar o PHP e vincular o endereço do meu site?

Não. No momento desativei esse recurso para evitar uso externos, você pode vincular usando o PHP, mas não recomendo isso, por questão de segurança.

Posso usar bibliotecas sob a licença GPL no meu software?

O MiPhant é um software compatível com a licença GPL, desde que você não distribua o executável do PHP junto com seu projeto. Use a versão original do software que baixa o PHP no computador do usuário, dessa forma evitando incompatibilidade com a licença.

Quero um recurso mais avançado para gerar PDF em PHP, qual utilizar?

Recomendamos a biblioteca DomPDF que está sob a licença LGPL, uma licença menor e compatível com softwares proprietários. Para softwares GPL você pode usar o DomPDF ou MPDF.

Como distribuir meu software?

Para distribuir seu software você pode criar um zip, AppImage, criar um instalador exe, entre outras formas. Defina a pasta de PHP com permissão de escrita e leitura para que seja possível baixar uma versão automaticamente. Coloque sempre uma licença no seu software, caso não saiba criar uma licença para seu software proprietário ou utilizar uma existente para software livre recomendamos sempre consultar um advogado.

Posso usar em um ambiente local?

Sim. O software é desenvolvido para funcionar com o servidor embutido do PHP em um ambiente local utilizando o IP 127.0.0.1, as portas são geradas automaticamente e de forma aleatória. Por esse motivo, nenhum computador externo terá acesso ao servidor, exceto se você configurar para que o IP local seja ouvido pela rede, o que não recomendamos, pois o IP local deve ser utilizado apenas para uso do usuário local, sem acesso a rede.

Posso usar em um computador público?

Evite utilizar o MiPhant em computadores públicos, pois ele utiliza o servidor embutido do PHP, exceto se realizar algum tratamento e aplicar técnicas de segurança, ao utilizá-lo, restrinja-se a servidores locais e configure o firewall para limitar o acesso apenas a endereços confiáveis.

Porque o software é compactado em asar?

Para evitar alterações acidentais nos arquivos principais do software.

Nome do Aplicativo

O nome do aplicativo tanto para Linux quanto para Windows foi alterado para maiúsculo, ou seja, de miphant para MiPhant, dessa forma você pode identificar o executável com mais facilidade.

Para distribuir seu aplicativo recomendamos que renomeie para o nome do seu aplicativo, exemplo de MiPhant para MeuApp ou MeuApp.exe.

Ícone da Janela

O ícone do aplicativo deve ser adicionado dentro da pasta app/icon/ com o nome que você definiu no arquivo config.json.

Caso não seja encontrado um ícone nessa pasta, será utilizado o ícone padrão do MiPhant.

Ícone do Aplicativo

O MiPhant para Windows possui um ícone próprio e exclusivo para identificar o executável com mais facilidade.

Tradução

Para que o software funcione corretamente é necessário criar o arquivo en.json, o idioma inglês é definido como padrão no MiPhant. Mesmo que seu software não tenha tradução, é necessário que o arquivo en.json seja criado, você pode deixá-lo sem itens, mas lembre-se de abrir e fechar as chaves {} para que o arquivo json fique no formato correto.

{}

Para criar as traduções você precisa definir seu aplicativo no idioma inglês e depois criar os outros idiomas, por exemplo, en.json, pt.json, es.json.

Exemplo en.json

{
"File":"File",
"Edit":"Edit",
"Help":"Help"
}

Exemplo pt.json

{
"File":"Arquivo",
"Edit":"Editar",
"Help":"Ajuda"
}

Menu

O software também pode ter menus principais, veja um exemplo de menu.

Environment

Através das variáveis de ambiente você pode fazer com que o script PHP obtenha algumas informações que são criadas no MiPhant.

  • MIPHANT_LANG: Obtém o idioma principal do sistema (pt, en, es)
  • MIPHANT_ARGV: Obtém os argumentos
  • MIPHANT_USERNAME: Obtém o nome de usuário
  • MIPHANT_HOMEDIR: Obtém a pasta home do usuário (/home/nomedousuario/ ou C:\Users\nomedousuario)
  • MIPHANT_PLATFORM: Obtém a plataforma do sistema (linux ou win32)
